## 🤖 Development Approach (Vibe Coding)

This project was built using a method called **Vibe Coding**.

* Instead of writing all code manually, I used AI tools like **OpenAI Codex / ChatGPT**
* I described features in natural language, and AI generated the code
* I tested, modified, and fixed errors based on outputs

Vibe Coding means:

* You explain what you want
* AI writes the code
* You run it and fix issues step by step

⚠️ Because of this approach:

* Some parts of the code may not be optimized
* Bugs and unexpected issues may exist
* Code structure may not follow strict best practices

This project is part of my learning journey using AI-assisted development.

# 📰 Modern News App (React Native + Expo)

A modern, fast, and minimal mobile news application built using **React Native (Expo)** with a focus on clean UI/UX, smooth performance, and real-time news updates.

---

## 🚀 About the Project

This app provides:

* 📱 Clean and modern news reading experience
* 🌙 Dark & Light mode support
* 🔊 Audio news (Text-to-Speech)
* 🔍 Search functionality
* 🌍 India & International news
* 🤖 AI-based news explanation

⚠️ **Note:**
This project was built while learning and experimenting with AI-assisted coding tools. Some parts of the code were generated using AI, so there may be bugs or incomplete features.

## 📦 Features

### 🏠 Home Screen

* Latest news (India & World)
* Category-based sections
* Smooth scrolling

### 🔍 Search

* Search news by keyword
* Dynamic results

### 📄 News Detail

* Full article view
* Image, title, source, time
* "Listen News" feature

### 🔊 Audio Mode

* Text-to-Speech using Expo Speech
* Continuous “Radio Mode” (auto-play news)

### 🤖 AI Explain

* Summarizes news in 50–60 words

* View news by date
* Navigate between days

* Bookmark news

## ⚙️ Installation & Setup

### 1. Clone Repository

```bash
git clone https://github.com/your-username/news-app.git
cd news-app
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Start App

```bash
npx expo start
```

### 4. Run on Device

* Install **Expo Go** app on your phone
* Scan QR code

---

## 🔑 Environment Variables

Create a `.env` file:
